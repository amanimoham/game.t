# محاربة المغريات — Frontend (React + Vite)

واجهة الويب للعبة. تتصل بالـ backend عبر `/api` (proxy إلى `http://localhost:8000`).

## التشغيل
```bash
npm install
npm run dev        # http://localhost:5173
```
تأكد أن الـ backend شغّال على المنفذ 8000 (docker compose up في مجلد backend).

## البناء
```bash
npm run build      # ينتج dist/
npm run preview
```

## الشاشات
- **دخول/تسجيل الأب** — إنشاء حساب أو تسجيل دخول.
- **لوحة الأب** — إضافة طفل، تعيين رقم سري، شراء وقفل مكافأة، عرض الرؤى (مهارات، توقّع الإكمال).
- **اللعبة (الطفل)** — خزنة الكنز تُفتح تدريجياً، بطاقات الوحوش، القرارات، رسائل تشجيع، ومؤشرات الـ AI (الصعوبة/الوحش المقترح/توقّع الإكمال).
